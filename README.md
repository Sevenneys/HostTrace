# 🔍 HostTrace

**Network Reconnaissance & Analysis Tool**

HostTrace — многофункциональный сетевой сканер для обнаружения хостов, анализа открытых портов, идентификации операционных систем и демонстрации ARP-спуфинга в учебных целях.

---

## 🚀 Возможности

| Функция | Описание |
|---------|----------|
| **ARP-сканирование** | Обнаружение активных устройств в локальной сети |
| **Порт-сканирование** | Анализ открытых TCP/UDP портов на целевых хостах |
| **Определение ОС** | Идентификация операционной системы устройства по сетевым отпечаткам |
| **ARP Spoofing / MITM** | Демонстрация атаки "человек посередине" в изолированной лабораторной среде |

---

## 📋 Требования

- Python 3.8+
- Linux / macOS / Windows (с правами администратора)
- Установка зависимостей из `requirements.txt`

## ⚡ Быстрый старт

# Клонирование репозитория
git clone https://github.com/Sevenneys/HostTrace.git
cd HostTrace

# Установка зависимостей
pip install -r requirements.txt

# Запуск (требуются права root/sudo)
sudo python main.py

# ARP-скан локальной сети
python main.py --scan

# Порт-скан целевого хоста
python main.py --ports 192.168.1.1

# Определение ОС
python main.py --os 192.168.1.1

# ARP-spoofing (учебный режим)
python main.py --spoof 192.168.1.1 192.168.1.254
